Shallow Copy & Deep Copy

There are 2 types of array cloning: shallow & deep.

Shallow Copy

Shallow copy is a bit-wise copy of an object. A new object is created that has an exact copy of the values in the original object.
just the reference addresses are copied i.e., only the memory address is copied.it will affect the origin one

Deep Copy

A deep copy means actually creating a new array and copying over the values, since whatever happens to it will never affect the origin one.
