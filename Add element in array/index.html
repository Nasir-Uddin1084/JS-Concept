splice() method

The splice() method changes the contents of an array by removing
or replacing existing elements and/or adding new elements.

To add new elements with Array.splice(),
just set the deleteCount to zero and pass new items:

# Array.splice(position,0,new_element_1);

let colors1 = ['red', 'green', 'blue'];
colors1.splice(2, 0, 'purple');

Output : ["red", "green", "purple", "blue"]

# Array.splice(position,0,new_element_1,new_element_2,....);

let colors1 = ['red', 'green', 'blue'];
splice(1, 0, 'yellow', 'pink');

Output : ["red", "yellow", "pink", "green", "purple", "blue"]
